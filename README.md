# Jay Tools - Free Online Calculators & Utilities

A comprehensive collection of free online tools and calculators designed to make everyday calculations simple and accessible. Jay Tools provides a wide range of utilities for students, professionals, and anyone who needs quick, reliable calculations.

## 🌐 Live Demo

**🔗 Website**: [https://www.jay-tools.com](https://www.jay-tools.com)

Experience all the tools live on our website. No installation required - just visit and start using any calculator or utility tool instantly!

## 🚀 Getting Started

### Local Development

1. **Clone or Download** the project files
2. **Open** `index.html` in your web browser
3. **Navigate** through the tools using the menu or search functionality

### Deployment

1. **Upload** all files to your web server
2. **Ensure** proper file permissions
3. **Update** any absolute URLs in the code
4. **Configure** web server for proper MIME types
5. **Test** all functionality on the live server

### Customization

#### Adding New Tools
1. Create new HTML file in `tools/` directory
2. Use existing tools as templates
3. Add tool information to search data
4. Update navigation menus
5. Add to sitemap.xml

#### Styling Changes
1. Edit `css/custom.css` for global changes
2. Create tool-specific CSS files for individual tools
3. Follow existing naming conventions
4. Test on all device sizes
